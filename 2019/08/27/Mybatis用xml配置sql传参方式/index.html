7、使用${}

#{}与${}的区别:
{}拿到值之后，拼装sql，会自动对值添加单引号”
${}则把拿到的值直接拼装进sql，如果需要加单引号”，必须手动添加，==一般用于动态传入表名或字段名使用==，同时需要添加属性statementType=”STATEMENT”，使用非预编译模式。
